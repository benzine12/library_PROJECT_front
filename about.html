About This Project

Project Description

Book CRM is a comprehensive library management system designed to keep track of books, customers, and loans. This project was built to facilitate easy management of library operations by enabling seamless book tracking, customer management, and loan records.

Technologies Used
  • Frontend: Bootstrap 5, FontAwesome for icons, HTML5, and CSS3
  • Backend: Flask (Python) and SQLAlchemy ORM
  • Database: SQLite
  • APIs: Flask RESTful APIs
